# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the actual labels, states, or equivalent fields used in this repo's issue tracker.

| Skill role          | Tracker expression | Meaning                                  |
| ------------------- | ------------------ | ---------------------------------------- |
| `needs-triage`      | `needs-triage`     | Maintainer needs to evaluate this issue  |
| `needs-info`        | `needs-info`       | Waiting on reporter for more information |
| `ready-for-agent`   | `ready-for-agent`  | Fully specified, ready for an AFK agent  |
| `ready-for-human`   | `ready-for-human`  | Requires human implementation            |
| `wontfix`           | `wontfix`          | Will not be actioned                     |

When a skill mentions a role, such as "apply the AFK-ready triage label", use the corresponding tracker expression from this table.

Edit the `Tracker expression` column to match the vocabulary this project actually uses.
